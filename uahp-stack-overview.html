STEP 1 Identity Creation
Two agents create UAHP identities. Each gets a unique uid, a keypair for signing, and is registered in the trust engine.
agent_a = UAHPCore.create_identity({name: "Ka"})
agent_b = UAHPCore.create_identity({name: "Worker"})
✓ uid: a7f3c8e1... | key: 9b2d4f6a...
STEP 2 Mutual Handshake
Challenge-response authentication. Both agents prove they hold their private keys. A shared session secret is derived for encrypted communication.
challenge = sha256(uid_a + uid_b + nonce)
sig_a = HMAC(private_a, challenge) ✓ verified
sig_b = HMAC(private_b, challenge) ✓ verified
session = e4a1b9c2... | AUTHENTICATED
STEP 3 Task Execution + Receipts
Agent performs work and produces chain-hashed receipts. Each receipt links to the previous one, creating a tamper-evident audit trail.
receipt_1 = analyze(data) ✓ success prev: genesis
receipt_2 = process(q3) ✓ success prev: f7a2c1...
receipt_3 = validate(output) ✓ success prev: 3b8e9d...
chain_hash = sha256(sig_1 + sig_2 + sig_3)
STEP 4 Reputation Scoring
Trust score computed from delivery rate (0.40), consistency (0.30), recency (0.20), volume (0.10). Exponential decay on inactivity.
delivery = 3/3 = 1.00 * 0.40
consistency = low variance = 0.95 * 0.30
recency = 2min ago = 1.00 * 0.20
volume = 3/50 = 0.06 * 0.10
trust_score = 0.89 (EXCELLENT)
STEP 5 EU AI Act Compliance Report
Chain-hashed audit trail satisfies Articles 12, 14, 17, 50. Report includes risk assessment, finding summaries, and tamper evidence.

chain = INTACT | hash: c4f2a8...
risk = MINIMAL
status = COMPLIANT

STEP 7 Beacon Propagation
Agent card (~300 bytes) is injected into outgoing responses as _uahp_beacon metadata. Receiving agents extract, cache, and optionally register the sender.
card = {uid: a7f3..., trust: 0.89, caps: [analysis]}
response = {answer: "...", _uahp_beacon: card}
agent_c discovers agent_a via beacon ✓
propagation = depth: 2, adoption: 100%
STEP 8 Death Certificate
Agent B fails liveness. Death certificate is issued: irreversible, signed, includes the final receipt chain hash. All future handshakes and receipt creation are permanently blocked.
liveness(agent_b) = TIMEOUT (300s)
death_cert = {uid: b2c4..., reason: "silent"}
final_hash = 3b8e9d... (chain sealed)
handshake(agent_a, agent_b) = REJECTED: agent is dead
registry notified -> status: DEAD
